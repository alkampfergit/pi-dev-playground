# Verification checklist

When explicitly asked to turn an existing plan or review into verification checks:

1. Include the exact marker `SKILL-MARKER: CHECKLIST`.
2. Return three to five Markdown checkbox items based on that plan or review.
3. Do not create, edit, or delete files.
